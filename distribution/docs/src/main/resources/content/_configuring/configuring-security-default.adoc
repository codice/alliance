== {title}

Configure the Security Default plugin to set a minimum security marking on any unmarked data ingested into ${branding}.

. Navigate to the *${admin-console}*.
. Select the *${alliance-security}* application.
. Select *Configuration*.

See <<org.codice.alliance.catalog.plugin.defaultsecurity.DefaultSecurityAttributeValuesPlugin,Default Security Plugin>> for all possible configuration options.
